# Система управления библиотекой

Проект для 10 класса по информатике. Программа для управления домашней библиотекой с использованием SQLite.

## Функциональность

- Добавление новых книг
- Просмотр каталога книг
- Поиск по автору или названию
- Отслеживание статуса прочтения
- Система рейтинга книг
- Статистика по библиотеке

## Технологии

- Python 3.x
- SQLite3
- Объектно-ориентированное программирование
